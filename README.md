Installation Instructions:

1. Unzip the attached file and open the extracted folder
2. Double-click "Victaulic-Vortex-Project-Configurator-Setup-v2.x.x.exe"
   This installation may take 15-30 seconds with no progress bar - this is normal.
3. Installation is complete when the green installer finishes and disappears. The app will launch automatically
4. Pin the configurator to your taskbar by right clicking the icon while it is open.

If you need to find the application folder:

1. Click Windows Key + R to open the "Run" command.
2. Paste in: %LocalAppData%\VictaulicVortexProjectConfigurator
